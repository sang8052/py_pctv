# 项目简介
这是一个基于 Python 开发的 个人PC端电视项目。目前仅可用于观看五星体育的直播
环境需求:

1. python >= 3.8


V1.1 版本  
1.移除 Redis 数据库依赖    
2.修复 m3u8 的密钥在播放一段时间后失效的问题


# 一、在浏览器中观看
项目默认使用 9655 端口，项目启动后你可以打开 http://localhost:9655 观看视频直播。
